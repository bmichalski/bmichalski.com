# bmichalski.com

## Start working
```sh
git clone git@github.com:bmichalski/bmichalski.com.git && \
git clone git@github.com:bmichalski/bmichalski.github.io.git && \
cd bmichalski.com
```

## Install vendors
```sh
yarn install
```

## Build
```sh
yarn run build-all
```

## Deploy
```sh
cd bmichalski.github.io.git && \
git push
```
